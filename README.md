# homebrew-qsys

Homebrew tap for Q-SYS-related tools.

## Install

```sh
brew tap reowens/qsys
brew install --cask qsys-mac-installer
```

`qsys-mac-installer` installs the signed, notarized Q-SYS Mac Installer app.
It does not include Q-SYS Designer; you provide your own Q-SYS Designer Windows
installer from QSC when you run the app.
